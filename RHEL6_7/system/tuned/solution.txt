The content stores tuned custom profiles located at /etc/tune-profiles to POSTUPGRADE_DIR.

After an upgrade to RHEL 7 the profile can be migrated manually with the help of commands 
tuned-man and tuned-profiles to directory /etc/tuned.
There is also a web page which describes tuned [link:http://docs.fedoraproject.org/en-US/Fedora/20/html/Power_Management_Guide/tuned.html].
